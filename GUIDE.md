**Prerequisites:**

- Ubuntu 18.04 or higher
- git
- golang 1.12.9
- tmux
- curl

**Minimum hardware requirement:**

- 8 CPU ~ 2.4Ghz
- 12 GB of RAM

**recommended hardware requirement:**

- 16 CPU ~ 2.4Ghz
- 24 GB of RAM

## #SECTION III: DEPLOY THE INCOGNITO CLI
## Installation
Install to the `$GOPATH` folder.
```shell
$ go install
```

## Build
1. Build Incognito binary file:

```
go build -o .
```
